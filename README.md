# 会动的小黄人 
## 介绍
会动的小黄人，是我在学习 CSS3 时使用 HTML5 + CSS3 + JavaScript 实现的一个小项目 
## 开始使用
```bash
git clone git@github.com:xuhaooo/minion.git 
cd minion
yarn global add http-server
# 或者 npm install -g http-server
hs src -c-1
```
## 功能 
- 支持动态的小黄人实现 
- 支持播放与暂停  
- 支持慢速与快速播放
